# Database Password Reset Steps

## ✅ Quick Steps:

1. **"Reset database password" button click karo**
   - Database Settings page par
   - "Database password" section mein

2. **Naya password set karo:**
   - Strong password use karo
   - Example: `MyListingAds2024!@#`
   - **Important:** Password save karo (notepad mein copy karo)

3. **Password copy karo:**
   - Reset ke baad password dikh jayega
   - Ya email mein aayega

4. **Backend .env update karo:**
   - `backend/.env` file kholo
   - `DB_PASSWORD=YOUR_SUPABASE_PASSWORD_HERE` 
   - Replace with: `DB_PASSWORD=MyListingAds2024!@#` (apna password)

5. **Backend restart karo**

---

## Alternative: Connection String Kahan Hai?

Agar URI tab nahi mil raha, try karo:

1. **Top bar mein "Connect" button:**
   - Dashboard top bar mein "Connect" button click karo
   - Wahan connection details mil sakte hain

2. **SQL Editor:**
   - Left sidebar → "SQL Editor"
   - Wahan bhi connection info mil sakta hai

3. **Project Overview:**
   - Left sidebar → "Project Overview"
   - Connection details wahan bhi ho sakte hain
